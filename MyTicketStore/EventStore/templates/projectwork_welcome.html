About the Project

This project demonstrates the implementation of Event Sourcing and CQRS patterns using Python.

Event Sourcing

Event Sourcing ensures that all changes to application state are stored as a sequence of events. This allows for better traceability and auditability.

CQRS

Command Query Responsibility Segregation (CQRS) separates the read and write operations, allowing for more scalable and maintainable code.
